## 1. Класс `WineDataModule`

Этот класс отвечает за подготовку данных и создание загрузчиков (`DataLoader`).
Он берёт обучающую и тестовую выборки, делит обучающую часть на train и validation,
а затем создаёт три источника данных для модели:

- **train_dataloader()** — подаёт батчи данных для обучения;
- **val_dataloader()** — используется для проверки качества модели после каждой эпохи;
- **test_dataloader()** — даёт данные для финальной оценки.

## 2. Класс `NeuralNetLightning`

Это сама модель, оформленная по правилам PyTorch Lightning.
Я реализовал в ней:

- **`__init__`** — создал простую нейросеть: вход 13 признаков, скрытый слой на 64 нейрона и выход на 3 класса;
- **`forward`** — описал, как данные проходят через слои сети;
- **`training_step`** — что происходит на каждом шаге обучения (расчёт ошибки и точности);
- **`validation_step`** и **`test_step`** — как считать качество на валидации и тесте;
- **`configure_optimizers`** — указал, что используется оптимизатор Adam с learning rate 0.001.
